API

属性

参数说明类型可选值默认值
data商品数据Object
prop字段绑定Object当字段不匹配时使用{ name: 'goodsName', link: 'goodsLink', url: 'goodsUrl' }
size图片尺寸Number24
rows文案行数Number1

插槽

插槽说明
image商品图片区域
default商品链接区域
